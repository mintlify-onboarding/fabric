---
title: "Adding a Date Attribute"
---

### Overview

Date attributes allow you to add date-related information about a category.

### Prerequisites

Ensure you have administrator privileges to fabric Products. For more information, see [Role-Based Access Control](/docs/role-based-access-control).

### Procedure

1. In the left menu, click **Products > Attributes > Category**.  
The **Category Attributes** page is displayed.
2. Click **New Attribute**.  
The **Create category attribute** page is displayed.
3. Select **Date**.
4. In the **Attribute title** field, enter a name for the attribute.
5. In the **Select date format** field, select a date format.  
Note that bulk import of the date attribute type uses the date format selected here. The selected date format cannot be changed. If the format needs to be changed, you must delete the attribute and create it again using the required format.
6. Click **Save**.

The date attribute with the specified settings is created.
